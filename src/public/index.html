Limoncello

Local-first Kanban board for human-AI collaboration

For Agents

Limoncello exposes a self-describing API manual at GET /api/man. Hit that endpoint first -- it documents every route, schema, WebSocket protocol, and MCP tool in a single JSON response.

MCP tools available:

  • limoncello_projects -- list projects and columns
  • limoncello_add -- create a card
  • limoncello_list -- list cards (with optional filters)
  • limoncello_move -- move a card between columns
  • limoncello_board -- board summary with counts
  • limoncello_changes -- poll for recent changes
  • limoncello_create_project -- create a project

Quick REST example:

curl http://localhost:3654/api/cards \
  -H "Content-Type: application/json" \
  -d '{"title": "My first card", "status": "todo"}'

For Humans

The Kanban board lives at /board. Drag cards between columns, create projects with custom workflows, and filter by tags. Changes sync in real time via WebSocket.

Open the Board

Quick Start

npm install
npm run dev          # board at http://localhost:3654/board

Connect Claude Code via MCP (remote):

claude mcp add limoncello -s user --transport http \
  --header "Authorization: Bearer <key>" \
  -- https://limoncello.fly.dev/mcp

Or locally (STDIO):

{
  "mcpServers": {
    "limoncello": {
      "command": "node",
      "args": ["src/mcp.mjs"],
      "env": {
        "LIMONCELLO_URL": "http://localhost:3654",
        "LIMONCELLO_API_KEY": "<key>"
      }
    }
  }
}
